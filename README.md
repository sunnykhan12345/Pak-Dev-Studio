# PakDevStudio Landing Page - Complete Next.js Project

This is a complete runnable Next.js project, not only component files.

## Run locally

```bash
npm install
npm run dev
```

Then open:

```txt
http://localhost:3000
```

## Included

- Next.js App Router
- JavaScript only
- Tailwind CSS v4
- Framer Motion 3D/tilt animations
- Dark theme default + light theme toggle
- Responsive sections
- SEO metadata, sitemap, robots
- All required public images/icons included

## Font note

Add licensed Nexa font files here if you have them:

```txt
/public/fonts/NexaProTrial-Regular.woff2
/public/fonts/NexaProTrial-Bold.woff2
```

The site will still run with fallback fonts if these font files are not present.
